Detailed Description

To run a Fast Marching Method on 2-D structured uniform grids.

This class enables running a Fast Marching procedure to calculate the signed distance function and extend a given front speed.

[in]gInstance of class Grid
[in]lsVector containing the level set function at grid nodes. The values are 0 on the interface (from which the distance is computed), positive on one side and negative on the other side. They must contain the signed distance on the nodes surrounding the interface but can take any value on other nodes, provided they have the right sign.
